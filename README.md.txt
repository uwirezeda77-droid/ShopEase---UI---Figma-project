## 1. Introduction

ShopEase is a mobile e-commerce application prototype designed in Figma, offering a simple and intuitive shopping experience — browsing products, searching, viewing product details, managing a cart, checking out, and receiving order confirmation. The design follows modern UI/UX principles: consistency, readability, accessibility, and user-friendly navigation.

## 2. Problem Statement

Many existing online shopping apps suffer from complex navigation, cluttered interfaces, and lengthy checkout processes. Customers often struggle to find products quickly, compare items, and complete purchases efficiently — reducing satisfaction and discouraging transactions.

ShopEase solves this with a clean, intuitive, mobile-first interface that simplifies browsing, shopping, and checkout.

## 3. Project Objectives

**General Objective:** Design a modern, user-friendly mobile e-commerce application prototype using Figma.

**Specific Objectives:**
- Design an attractive and consistent mobile interface

- Create an efficient shopping experience from product selection to checkout
- Apply modern UI/UX design principles
- Improve accessibility and usability

## 5. User Flow

```
Splash Screen
   ↓
Login
   ↓
Home Screen
   ↓
Product Details
   ↓
Shopping Cart
   ↓
Checkout
   ↓
Order Success
```

This flow represents the complete shopping journey from launching the app to successfully placing an order.

## 6. Wireframes

Low-fidelity wireframes were created before the final UI to define screen structure, navigation flow, component placement, and user interaction. See `/wireframes` folder.

## 7. High-Fidelity UI Design

Seven fully designed screens were built in Figma:

1. **Splash Screen** – app branding, welcome message, navigation to Login
2. **Login Screen** – email field, password field, login button, Google sign-in
3. **Home Screen** – welcome message, search bar, categories, featured products, bottom nav
4. **Product Details Screen** – product image, name, price, description, size/color selection, Add to Cart
5. **Shopping Cart Screen** – product list, quantity controls, price summary, checkout button
6. **Checkout Screen** – shipping address, payment method, order summary, place order button
7. **Order Success Screen** – success message, confirmation icon, continue shopping button

See `/high-fidelity-designs` folder for exported screens.

## 9. Accessibility Considerations

- Readable font sizes throughout
- High contrast between text and background
- Consistent button styles and navigation
- Clear labels and icons
- Proper spacing between elements
- Easy-to-understand layouts
- Consistent color scheme across all screens
